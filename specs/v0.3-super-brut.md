# v0.3 -- Super brut / cout employeur

## Resume

La v0.3 ajoute le calcul du **super brut** (cout total employeur) pour les salaries du secteur prive. Le visiteur peut desormais visualiser la chaine complete : super brut --> salaire brut --> net avant impot --> net apres IR. Le moteur calcule les cotisations patronales obligatoires a partir du salaire brut et applique la reduction generale degressive unique (RGDU, ex-Fillon) lorsqu'elle est applicable. Les taux utilises sont ceux en vigueur au 1er janvier 2026.

## User Stories

### US-300 : Visualiser le cout total employeur (super brut)

> En tant que visiteur, je veux voir le cout total employeur (super brut) correspondant a mon salaire brut afin de comprendre combien mon emploi coute reellement a mon entreprise.

**Criteres d'acceptation :**

- [ ] Le super brut est calcule automatiquement a partir du salaire brut saisi
- [ ] Le detail des cotisations patronales est affiche (chaque ligne avec libelle, assiette, taux, montant)
- [ ] Le total des cotisations patronales est affiche
- [ ] La RGDU (si applicable) est affichee comme une ligne de reduction negative
- [ ] Le super brut final (brut + cotisations patronales - RGDU) est affiche
- [ ] La chaine complete super brut --> brut --> net avant impot --> net apres IR est visible

### US-301 : Configurer les parametres employeur

> En tant que visiteur, je veux pouvoir ajuster les parametres specifiques a mon entreprise (effectif, taux AT/MP, versement mobilite) afin que le calcul reflete ma situation reelle.

**Criteres d'acceptation :**

- [ ] Un selecteur de tranche d'effectif est disponible (< 11 / 11-49 / >= 50) avec la valeur par defaut >= 50
- [ ] Le taux AT/MP est pre-rempli a 2,08 % (taux moyen national 2026) et modifiable
- [ ] Le versement mobilite est activable via une case a cocher, avec un taux par defaut de 2,95 % (taux Paris) modifiable
- [ ] Le taux de prevoyance cadres est pre-rempli a 1,50 % (minimum conventionnel) et modifiable (visible uniquement si statut cadre)
- [ ] Les modifications de parametres recalculent immediatement les resultats

### US-302 : Comparer le super brut entre deux situations

> En tant que visiteur, je veux comparer le cout employeur entre ma Situation 1 et ma Situation 2 afin de visualiser l'impact d'un changement de salaire sur le cout total.

**Criteres d'acceptation :**

- [ ] Chaque situation dispose de ses propres parametres employeur
- [ ] Le delta (difference) est affiche sur le super brut, les cotisations patronales totales, et la RGDU
- [ ] Les deltas positifs et negatifs sont visuellement distingues

### Calcul du super brut

**RM-216 -- Formule du super brut**
`super_brut = salaire_brut + total_cotisations_patronales - rgdu`

Le super brut represente le cout total employeur. La RGDU vient en deduction des cotisations patronales.

**RM-217 -- Arrondis**
Les montants de cotisations sont calcules avec une precision de 2 decimales (centimes d'euro). Le super brut final est arrondi au centime.

### Reduction Generale Degressive Unique (RGDU)

**RM-218 -- Eligibilite RGDU**
La RGDU s'applique aux salaries dont la remuneration annuelle brute est inferieure a **3 SMIC annuels** (65 629,20 EUR en 2026). Au-dela, aucune reduction n'est applicable.

**RM-219 -- Formule du coefficient de reduction**
```
coefficient = T_min + (T_delta * ((1/2) * (3 * SMIC_annuel / remuneration_brute_annuelle - 1)) ^ P)
```

Parametres 2026 :
- `T_min` = 0,0200
- `P` = 1,75
- `T_delta` : voir RM-220

**RM-220 -- Valeur de T_delta selon l'effectif**

| Effectif | T_delta | T_max (= T_min + T_delta) |
|----------|---------|---------------------------|
| < 50 salaries | 0,3781 | 0,3981 |
| >= 50 salaries | 0,3821 | 0,4021 |

`T_max` represente le coefficient maximal de reduction, atteint lorsque la remuneration est egale au SMIC.

**RM-221 -- Arrondi et plafonnement du coefficient**
Le coefficient est arrondi a **4 decimales** (au dix millieme le plus proche).
Le coefficient ne peut pas depasser `T_max` (= T_min + T_delta).
Le coefficient ne peut pas etre negatif. Si le calcul donne un resultat negatif, le coefficient est de 0.

**RM-222 -- Montant de la RGDU**
```
rgdu = remuneration_brute_annuelle * coefficient
```

La RGDU ne peut pas depasser le total des cotisations patronales eligibles. Si le montant calcule de la RGDU depasse le total des cotisations eligibles, la RGDU est plafonnee a ce total.

**RM-223 -- Cotisations couvertes par la RGDU**
La RGDU s'impute sur les cotisations patronales suivantes :
- Assurance maladie (13,00 %)
- Vieillesse plafonnee (8,55 %)
- Vieillesse deplafonnee (2,11 %)
- Allocations familiales (5,25 %)
- AT/MP (fraction mutualisee de 0,49 % uniquement)
- CSA (0,30 %)
- FNAL (0,10 % ou 0,50 %)
- Assurance chomage (4,00 %)
- Agirc-Arrco T1 (4,72 %)
- CEG T1 (1,29 %)
- CET (0,21 %)

Note : les cotisations T2 (Agirc-Arrco, CEG) ne sont pas couvertes par la RGDU.

**RM-224 -- Repartition RGDU entre URSSAF et IRC**
Le coefficient est reparti entre les cotisations URSSAF et les cotisations versees aux institutions de retraite complementaire (IRC). La part IRC du coefficient ne peut pas depasser **6,01 %** (0,0601). Le reste s'impute sur les cotisations URSSAF.

**RM-225 -- Calcul simplifie (annuel)**
Dans le simulateur, le calcul de la RGDU est effectue sur une base annuelle simplifiee. Le calcul mensuel avec regularisation progressive (utilise en paie reelle) est hors perimetre.

## Formules de calcul

### Pseudo-code complet du calcul des cotisations patronales et du super brut

```
ENTREES:
  salaire_brut_annuel        : nombre >= 0
  is_cadre                   : boolean
  effectif                   : "moins_11" | "11_49" | "50_plus"
  taux_at_mp                 : nombre >= 0 (defaut : 0.0208)
  is_versement_mobilite      : boolean (defaut : false)
  taux_versement_mobilite    : nombre >= 0 (defaut : 0.0295)
  taux_prevoyance_cadres     : nombre >= 0 (defaut : 0.0150)

CONSTANTES:
  PASS = 48 060
  SMIC_ANNUEL = 21 876.40
  PLAFOND_4_PASS = 192 240
  PLAFOND_8_PASS = 384 480

--- ETAPE 1 : Calcul des assiettes ---

assiette_totalite = salaire_brut_annuel
assiette_TA = MIN(salaire_brut_annuel, PASS)
assiette_T2 = MAX(0, MIN(salaire_brut_annuel, PLAFOND_8_PASS) - PASS)
assiette_4_PASS = MIN(salaire_brut_annuel, PLAFOND_4_PASS)
assiette_8_PASS = MIN(salaire_brut_annuel, PLAFOND_8_PASS)

--- ETAPE 2 : Calcul des cotisations patronales URSSAF ---

maladie = assiette_totalite * 13.00 %
vieillesse_plafonnee = assiette_TA * 8.55 %
vieillesse_deplafonnee = assiette_totalite * 2.11 %
allocations_familiales = assiette_totalite * 5.25 %
at_mp = assiette_totalite * taux_at_mp
csa = assiette_totalite * 0.30 %

SI effectif = "50_plus":
  fnal = assiette_totalite * 0.50 %
SINON:
  fnal = assiette_TA * 0.10 %
FIN SI

chomage = assiette_4_PASS * 4.00 %
ags = assiette_4_PASS * 0.25 %

--- ETAPE 3 : Calcul des cotisations Agirc-Arrco patronales ---

agirc_arrco_t1 = assiette_TA * 4.72 %
agirc_arrco_t2 = assiette_T2 * 12.95 %
ceg_t1 = assiette_TA * 1.29 %
ceg_t2 = assiette_T2 * 1.62 %
cet = assiette_8_PASS * 0.21 %

SI is_cadre:
  apec = assiette_8_PASS * 0.036 %
SINON:
  apec = 0
FIN SI

--- ETAPE 4 : Cotisations complementaires ---

SI is_cadre:
  prevoyance = assiette_TA * taux_prevoyance_cadres
SINON:
  prevoyance = 0
FIN SI

SI is_versement_mobilite ET effectif != "moins_11":
  versement_mobilite = assiette_totalite * taux_versement_mobilite
SINON:
  versement_mobilite = 0
FIN SI

taxe_apprentissage = assiette_totalite * 0.68 %

SI effectif = "moins_11":
  cfp = assiette_totalite * 0.55 %
SINON:
  cfp = assiette_totalite * 1.00 %
FIN SI

--- ETAPE 5 : Total des cotisations patronales (avant RGDU) ---

total_cotisations_patronales = maladie
  + vieillesse_plafonnee + vieillesse_deplafonnee
  + allocations_familiales
  + at_mp
  + csa
  + fnal
  + chomage + ags
  + agirc_arrco_t1 + agirc_arrco_t2
  + ceg_t1 + ceg_t2
  + cet
  + apec
  + prevoyance
  + versement_mobilite
  + taxe_apprentissage
  + cfp

--- ETAPE 6 : Calcul de la RGDU ---

SI salaire_brut_annuel >= 3 * SMIC_ANNUEL:
  rgdu = 0
SINON SI salaire_brut_annuel <= 0:
  rgdu = 0
SINON:
  // Determiner T_delta et T_max selon l'effectif
  SI effectif = "50_plus":
    T_delta = 0.3821
  SINON:
    T_delta = 0.3781
  FIN SI

  T_min = 0.0200
  T_max = T_min + T_delta
  P = 1.75

  // Calcul du coefficient
  ratio = (1/2) * (3 * SMIC_ANNUEL / salaire_brut_annuel - 1)

  SI ratio <= 0:
    coefficient = 0
  SINON:
    coefficient = T_min + (T_delta * ratio ^ P)
    coefficient = ARRONDI_4_DEC(coefficient)  // arrondi au dix millieme
    coefficient = MIN(coefficient, T_max)
  FIN SI

  // Calcul du montant
  rgdu = salaire_brut_annuel * coefficient

  // Plafonnement : la RGDU ne peut pas depasser les cotisations eligibles
  // (cotisations couvertes, cf. RM-223)
  cotisations_eligibles_rgdu = maladie
    + vieillesse_plafonnee + vieillesse_deplafonnee
    + allocations_familiales
    + MIN(at_mp, assiette_totalite * 0.49 %)  // fraction mutualisee uniquement
    + csa
    + fnal
    + chomage
    + agirc_arrco_t1
    + ceg_t1
    + cet

  rgdu = MIN(rgdu, cotisations_eligibles_rgdu)
FIN SI

--- ETAPE 7 : Calcul du super brut ---

super_brut = salaire_brut_annuel + total_cotisations_patronales - rgdu

--- ETAPE 8 : Resultats ---

taux_cotisations_patronales = total_cotisations_patronales / salaire_brut_annuel * 100
taux_apres_rgdu = (total_cotisations_patronales - rgdu) / salaire_brut_annuel * 100
```

- Cotisations eligibles = 3 900 + 2 565 + 633 + 1 575 + 147 (AT/MP mut. 0,49 %) + 90 + 30 + 1 200 + 1 416 + 387 + 63 = 12 006,00 EUR
- rgdu (5 157,00) < cotisations eligibles (12 006,00) : pas de plafonnement

Super brut :
- 30 000 + 12 927,00 - 5 157,00 = **37 770,00 EUR**
- Taux de cotisations patronales (apres RGDU) : 25,90 %

## Hors perimetre

Les elements suivants sont explicitement exclus de la v0.3 :

- **Calcul mensuel RGDU avec regularisation progressive** : le simulateur utilise un calcul annuel simplifie (cf. RM-224)
- **Exonerations specifiques** : LODEOM, ZRR, ZFU, JEI, emplois a domicile, contrats aides
- **Regime Alsace-Moselle** : cotisation maladie salariale supplementaire (prevu dans idees futures)
- **Temps partiel** : prorata du SMIC et du PASS pour le calcul de la RGDU
- **Apprentis** : exoneration specifique de cotisations
- **CDD** : majoration de contribution assurance chomage ou taxe CDD
- **Forfait social** : contribution sur les elements de remuneration exoneres de cotisations (interessement, participation)
- **Versement mobilite regional et rural (VMRR)** : contribution regionale supplementaire creee par la LFI 2025
- **Prevoyance non-cadres** : variable selon convention collective, non integree
- **Contribution dialogue social** : 0,016 % patronal (montant negligeable)
- **Micro-entrepreneur** : prevu en v0.4
